# agilercloud/homebrew-tap

Homebrew tap for [agilerCloud](https://agiler.io) tools.

## Install

```sh
brew install agilercloud/tap/agiler
```

Running `brew install <tap>/<cask>` taps implicitly; no separate `brew tap` is required.

## What's here

| Cask | Source | Description |
|---|---|---|
| [`agiler`](Casks/agiler.rb) | [agilercloud/cli](https://github.com/agilercloud/cli) | Command-line interface for Agiler |

## Updating

Casks are regenerated automatically on every release of the corresponding source repo via GoReleaser. Do not edit files in `Casks/` by hand — changes will be overwritten on the next release.
